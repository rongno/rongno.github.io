可以写字了！

可以写字了。

首先看聊天记录

  1. 在E盘hexo目录下右键 Git Bish。

  2. 在$后输入英文状态下空格，再输入 hexo new post"201307211" 回车。

  3. 打开 Sublime Text ,找到hexo/source/_posts/201307211 ,开始写字。

  4. 写完后找到 Git Bish 的窗口，在 $ 后输入英文状态下空格，再输入 hexo server 回车。

  5. 在浏览器中输入 http://locolhost:4000/ 查看效果。

  6. ①如果没问题，继续在窗口中输入空格后，再输入 hexo generatr 回车，hexo deploy 回车，输入用户名、密码同步到网上。

    ②如果出现问题，在 Git Bish 窗口下 Ctrl+C 终止程序，返回 Sublime Text 中修改并保存，重复步骤4。

然后现在看下效果，哈哈哈。
