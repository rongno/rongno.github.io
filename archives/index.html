我

有一个自己的空间，可以没有别人，我都忘了是从多久以前开始有这样的想法，应该是从一个我躺在被窝的晚上开始，或者从我拿着日记找一个没人的地方开始，好像我就是想要断了各种联系，并做好准备接受一切。

我不会断了各种联系，没有任何关系，我只是有时候会把它们放在一个脆弱的环境来尝试，不知道自己是不是有任何期待。当准备接受一切时，任何结果已经没什么区别。就像我只在自己一个人时写日记，但写完了会随手放桌上，我不知道别人是不是看到了，我知道别人看了也不会跟我说什么。

我开始注意表达方式，而不是内容，有太多内容让人觉得枯燥乏味，它们只是表达方式的呈现，像被随手拿来把玩的玩意儿。

即使是这样想，我的心仍然不能平静，因为我没有做到早睡早起身体好！

7月 21 2013

可以写字了！

可以写字了。

首先看聊天记录

  1. 在E盘hexo目录下右键 Git Bish。

  2. 在$后输入英文状态下空格，再输入 hexo new post"201307211" 回车。

  3. 打开 Sublime Text ,找到hexo/source/_posts/201307211 ,开始写字。

  4. 写完后找到 Git Bish 的窗口，在 $ 后输入英文状态下空格，再输入 hexo server 回车。

  5. 在浏览器中输入 http://locolhost:4000/ 查看效果。

  6. ①如果没问题，继续在窗口中输入空格后，再输入 hexo generate 回车，hexo deploy 回车，输入用户名、密码同步到网上。

    ②如果出现问题，在 Git Bish 窗口下 Ctrl+C 终止程序，返回 Sublime Text 中修改并保存，重复步骤4。

然后现在看下效果，哈哈哈。
