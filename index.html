1001的字

很残忍，我想起了《枕头人》，竟然会有这么残忍的故事，这是真还是假，我甚至要怀疑每天晚上枕头都在给我讲这样残忍的事，不然我怎么会紧张。

今天算是10.1了吧，太快了，我一直在重复昨天的事情。又是凌晨，凌晨的东西总是很短暂，很急促。思想又停滞了，它已经睡了，可我总是担心来不及，二十五年很快过去了，听起来很长，可我觉得它像凌晨一样短暂，急促，没留下什么回忆就再也回不来了。

我应该感谢这些字，从来没问过它们是否愿意就这样出现，我让它们组成借口、理由，让我心安理得。也从其它的排列组合中感受到了惊喜和安慰。它们的出现好像代表了更重的责任，让人不敢轻易承担。

关于画图

在画的又开始反胃时，我想起了一个很美好的愿望。

我想开心的画图。

可能这个要求太高了，然后一直不想画，只是逃开等着心情变好，很多年过去了，蓦然回首，这方法竟然没用！记得上学时有次临摹一张室内的手绘，我找了一个绿色的儿童房，很开心的画完了，好像再也没有那么开心的画过，再也没有画那么好过。还有在宏村，一个人在里面晃荡半天，悠闲的找个没人的小角落，画一段幽静的胡同，或者绕到屋后面，画一面斑驳的红墙，或者在池塘边和当地的小朋友玩一会再画画正在泡澡的牛，只有这样画出来的画我才有勇气一直看着他们，也不知道是喜欢这幅画还是喜欢体会当时的心情。记得在做毕业设计时怎么也不想画图，停了几天有一天晚上突然发现灯光透过教学楼的窗户很美，然后意识到原来建筑是可以很美的，这样毕业设计才可以继续画下去。我好像一个昏昏沉沉记性很差的人，只能找些美好的事情刺激下，才能清醒下。

前几天看到康的《静谧与光明》，里面几次提到“道（order)”,觉得很惊喜，看到那一句"试想，人们可以拥有一小片阳光"，觉得很感动，看到他问“砖”想成为什么，觉得我把石头当人也不是很神经。看书真的会让人感觉不孤独，至少对我这样一个自私的不想和别人分享时间的人——听说水至清则无鱼——很多时候我是想画像清水那样的建筑，有时候别人想帮我放几条鱼时，在我看来是放了几只旱鸭子，看的我很揪心，不忍直视，也许我该学着在里面种点水草。

可以写字了！

可以写字了。

首先看聊天记录

  1. 在E盘hexo目录下右键 Git Bash here。

  2. 在$后输入英文状态下空格，再输入 hexo new post"201307211" 回车。

  3. 打开 Sublime Text ,找到hexo/source/_posts/201307211 ,开始写字。

  4. 写完后找到 Git Bssh here 的窗口，在 $ 后输入英文状态下空格，再输入 hexo server 回车。

  5. 在浏览器中输入 http://locolhost:4000/ 查看效果。

  6. ①如果没问题，继续在窗口中输入空格后，再输入 hexo generate 回车，hexo deploy 回车，输入用户名、密码同步到网上。

    ②如果出现问题，在 Git Bish 窗口下 Ctrl+C 终止程序，返回 Sublime Text 中修改并保存，重复步骤4。

然后现在看下效果，哈哈哈。
